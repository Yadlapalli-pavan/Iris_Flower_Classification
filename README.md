# 🌸 Iris Flower 

This project implements a machine learning classification system to identify different species of iris flowers using the famous Iris dataset. The dataset contains measurements of flower features, and multiple machine learning algorithms are used to predict the flower species.

**📊 Dataset**

The project uses the Iris Dataset, which is one of the most well-known datasets in machine learning.

**Dataset Features:**

    Sepal Length
    Sepal Width
    Petal Length
    Petal Width
    
**Target Classes:**

    Iris-setosa
    Iris-versicolor
    Iris-virginica
    
**⚙️ Technologies and Libraries Used**

    **Python**
    
    **Pandas** – Data manipulation
    
    **NumPy** – Numerical computations
    
    **Matplotlib** – Data visualization
    
    **Seaborn** – Statistical visualization
    
    **Scikit-learn** – Machine learning models and evaluation
